# Oinam Jekyll

A simple, clean, and minimal Jekyll Theme.

## Installation

Add this line to your Jekyll site's Gemfile:

```ruby
gem "oinam-jekyll"
```

Run

`bundle`

## Usage

Have the following line in your config file:

```yaml
theme: oinam-jekyll
```
